# About
This is a simple Quiz App designed to let users take quizzes on a variety of topics. It also allows admins to easily create and delete quizzes for users.
Built using TS, Express.js and EJS.

# Setup
Rename [.env.example](.env.example) to .env, configure values.

Install the [Bun](https://bun.sh/) runtime. Then, run:
```sh
bun run prod
```
The server will be active on `http://localhost:3000` (or whatever port you set it to)!
